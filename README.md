# Library of Lenses

*This is the place to learn about different vintage lenses and my experience with them. Some are obscure, others quite common â€“ from cult classics to stuff with zero information online, stop by and browse the library.*

This project is a replica of www.libraryoflenses.com, built from scratch. The main aim was to learn more about TypeScript, Three.js fiber and GSAP, as well as improve the admin experience, as opposed to wordpress and the many steps it took to post a review.

### Additional  Technologies:

- Zustand  - Used for state management.
- gltfjsx - to convert the 3D .glb file to a reusable component
- Three.js fiber - Used for the 3D models on the landing page
- Three.js drei - Used for scroll animation on the 3D models on the landing page
- gsap - Used to animate the 3D models on the landing page
- react-hook-form - Used to manage forms on login,register and review creation page
- yup - For schema validation on register form
